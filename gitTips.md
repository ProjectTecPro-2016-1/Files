Baixar repositório novo:
	git clone e_o_link_do_git

Para visualizar as modificações dos arquivos:
	git status

Para confirmar as alterações dos arquivos:
	git add nome_do_arquivo1 nome_do_arquivo2 ...

Para dar um commit (comentar o que foi feito):
	git commit -m"MENSAGEM"

Para enviar para o repositório online:
	git push origin master

Para baixar os arquivos mais atuais do repositório online:
	git pull origin master
